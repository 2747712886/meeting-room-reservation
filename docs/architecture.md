# 系统架构设计

## 架构概览

```text
浏览器
  |
Vue 3 前端
  |
Spring Boot 后端
  |---- MySQL：存储用户、会议室、预约、日志
  |---- Redis：缓存会议室详情、可预约时间、用户权限
  |---- RabbitMQ：异步通知
  |---- Spring Task：过期预约处理、会议提醒
```

## 核心链路

1. 用户登录后获取 JWT。
2. 前端携带 JWT 请求后端接口。
3. Spring Security 校验身份和权限。
4. 预约接口先获取 Redisson 分布式锁。
5. 后端在事务中检测预约时间冲突并写入预约记录。
6. 事务成功后发送 RabbitMQ 消息。
7. 消费者异步生成通知记录。
